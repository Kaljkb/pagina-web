Amenazas Digitales Más Comunes

🎣 Phishing

Intento de engaño mediante correos electrónicos, mensajes o sitios web falsos para obtener información confidencial como contraseñas, datos bancarios o información personal.

¿Cómo identificarlo?
  • Remitente desconocido.
  • Errores ortográficos.
  • Enlaces sospechosos.
Cómo protegerte?
  • Verifica la dirección web.
  • No compartas contraseñas.
  • Activa la verificación en dos pasos.

🦠 Malware

Software malicioso diseñado para dañar dispositivos, espiar actividades o robar información sin autorización del usuario.

¿Cómo identificarlo?
  • Equipo lento.
  • Ventanas emergentes extrañas.
  • Programas desconocidos instalados.
¿Cómo protegerte?
  • Instala un antivirus.
  • Descarga solo de sitios oficiales.
  • Mantén actualizado el sistema.

🔒 Ransomware

Tipo de malware que bloquea archivos o sistemas completos y exige un pago para recuperar el acceso a la información.

¿Cómo identificarlo?
  • Archivos inaccesibles.
  • Mensajes solicitando dinero.
  • Cambio repentino de extensiones.
¿Cómo protegerte?
  • Realiza copias de seguridad.
  • No abras archivos sospechosos.
  • Actualiza tus programas.

🎭 Ingeniería Social

Técnica que manipula psicológicamente a las personas para obtener información confidencial o acceso a sistemas.

Identificarla?
  • Solicitudes urgentes.
  • Promesas demasiado buenas.
  • Petición de datos personales.
¿Cómo protegerte?
  • Verifica la identidad del solicitante.
  • No compartas información sensible.
  • Desconfía de mensajes inesperados.

🆔 Robo de Identidad

Uso indebido de información personal para hacerse pasar por otra persona y realizar actividades fraudulentas.

¿Cómo identificarlo?
  • Movimientos financieros extraños.
  • Cuentas desconocidas a tu nombre.
  • Correos de servicios no registrados.
¿Cómo protegerte?
  • Utiliza contraseñas seguras.
  • No publiques datos personales.
  • Activa la autenticación multifactor.

Consejos de Prevención

  • Verificar siempre la dirección web antes de ingresar datos.
  • No abrir enlaces sospechosos.
  • Mantener actualizado el sistema operativo.
  • Utilizar contraseñas seguras.
  • Activar la autenticación en dos factores.
  • Instalar software de seguridad confiable.
